⚙ SDKs
Ship payments in 5 minutes
Your agent can send, receive, and track money with 3 API calls.
1
Install the SDK — npm install bankrails
2
Create an agent wallet — one line, returns a wallet ID
3
Trigger a payment — agent pays or gets paid, webhook fires on settlement
// Your agent can pay in 3 lines import { BankRails } from 'bankrails' const br = new BankRails(process.env.BR_API_KEY) // Create a wallet for your agent const wallet = await br.agents.create({ name: 'research-agent-01', budget: 50.00 // hard limit }) // Agent pays for a completed task await br.payments.send({ from: wallet.id, to: 'acct_freelancer_42', amount: 12.50, memo: 'SEO article — 1200 words' }) // → { id: 'pay_9kXm', status: 'processing' }
